# 리액트 프로 

## 리액트 정의 
- 리액트는 엔진이다. 
- 조합형 사용자 인터페이스를 구축한다. 
- 자바스크립트와 xml을 이용한다. 


## 리액트의 장점 
### 편리한 반응형 렌더링 
1. 새로운 페이지 전송 
2. DOM일부 변경 
3. 바인딩 
4. 렌더링 
5. 가상 DOM

### 순수 자바스크립트를 이용한 컴포넌트 기반 개발

### 문서 모델의 유연한 추상화 
UI의 경량 표준을 갖추고 있어서 HTML은 물론 IOS, android 뷰를 렌더링할 때도 동일한 원칙이 적용된다. 

### 리액트 개발 워크플로
- JSX를 작성하고 즉석에서 일반 자바스크립트로 변환
- 코드를 모듈 패턴으로 작성 
- 의존성 관리 
- 자바스크립트 파일을 번들로 만들고 디버깅 위해 소스 맵을 사용
